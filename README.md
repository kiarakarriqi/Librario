
📚 Librario – Library Management System

Librario is a web-based library management system designed to simplify the process of borrowing, 
reserving, and managing books. The system improves user experience, enhances library staff efficiency, 
and gives administrators full control over the library operations.



👥 User Roles

1️⃣ Administrator

	•	Manage all user accounts (members and staff)
	
	•	Add, edit, or remove books, authors, and categories
	
	•	Configure fine policies for overdue books
	
	•	Monitor all borrowings and reservations
	
	•	Generate reports and view system statistics
	

2️⃣ Library Staff

	•	Handle borrowing requests from members
	
	•	Confirm and process book loans
	
	•	Manage book check-outs and check-ins
	
	•	Update book availability
	
	•	Notify members about overdue fines
	

3️⃣ Member

	•	Create and manage a personal account
	
	•	Browse books by title, author, or category
	
	•	View book details and real-time availability
	
	•	Reserve and borrow books
	
	•	Track borrowing history and fines
	

⸻

🔑 Key Features

	•	Role-based access control
	
	•	Real-time book availability
	
	•	Borrowing and reservations management
	
	•	Fine calculation for overdue books
	
	•	Library catalog management
	
	•	Notifications and reporting
